- `watch` 不是 npm 自帶魔法，通常只是你自訂的 script 名稱，背後常用 `nodemon`


假設我只要指定執行某兩行js 最快的方式是甚麼 簡單說

```powershell
node -e "const arr2 = Array.from({ length: 5 }, (_, i) => i + 1); console.log(arr2)"
```

如果只是臨時測 1~2 行，這是最快的。

補一句區別：

- 要跑整個檔案：`node .\集合物件\array.js`
- 只想測幾行：`node -e "..."`
- 想互動試寫：直接輸入 `node` 進 REPL 再一行行打

你這種情境，**最短答案就是 `node -e`**。
