## 正規表達式特殊符號（Pattern 語法元素）完整總表

| 特殊符號 | 說明 | 範例與比對結果 | 字元解說 | 分類 |
|:---:|:---|:---|:---|:---|
| `+` | 一次或以上 | `/go+d/` → 可搜尋到 god, good, goood 等字串 | `/go+d/` // `g` 匹配 "g"，`o+` 匹配 o 一次或多次，`d` 匹配 "d" | 量詞 |
| `?` | 零次或一次 | `/colou?r/` → 可同時搜尋到 color 或 colour 字串 | `/colou?r/` // `color` 或 `colour`，`u?` 表示 u 可有可無 | 量詞 |
| `*` | 零次或多次 | `/go*d/` → 可搜尋到 gd, god, good 等字串 | `/go*d/` // `g` 匹配 "g"，`o*` 匹配 o 零次或多次，`d` 匹配 "d" | 量詞 |
| `{n}` | 出現剛好 n 次 | `/a{2}/` → 搜尋 a 連續出現 2 次的狀況 | `/a{2}/` // `a{2}` 匹配連續兩個 a | 量詞 |
| `{n,}` | 出現 n 次或以上 | `/a{2,}/` → 搜尋 a 出現 2 次（含）以上的狀況 | `/a{2,}/` // `a{2,}` 匹配 a 出現 2 次或更多次 | 量詞 |
| `{n,m}` | 出現 n 到 m 次 | `/a{1,3}/` → 搜尋 a 出現 1 到 3 次的狀況 | `/a{1,3}/` // `a{1,3}` 匹配 a 出現 1~3 次 | 量詞 |
| `*?` | 惰性匹配（0次或多次） | `/<.*?>/` → 匹配最短的 `<>` 區間 | `/<.*?>/` // `<` 匹配 "<"，`.*?` 任意字元盡量少，`>` 匹配 ">" | 量詞（惰性） |
| `+?` | 惰性匹配（1次或多次） | `/<.+?>/` → 匹配最短的 `<>` 區間 | `/<.+?>/` // `<` 匹配 "<"，`.+?` 任意字元至少一次且盡量少，`>` 匹配 ">" | 量詞（惰性） |
| `\|` | 用「或」(or) 的方式找出字串 | `/company(y\|ies)/` → 可搜尋到 company 或 companies | `/company(y\|ies)/` // `company` 後接 `y` 或 `ies` | 群組與條件 |
| `( )` | 表示集合（分組） | `/s(cc)d/` → 表示中間有一個以上的 "cc" 字串 | `/s(cc)d/` // `s` 匹配 "s"，`(cc)` 捕獲 "cc"，`d` 匹配 "d" | 群組與條件 |
| `(?: )` | 非捕獲組 | `/(?:\d+)/` → 匹配數字但不儲存 | `/(?:\d+)/` // `(?: )` 非捕獲分組，`\d+` 匹配數字 | 群組 |
| `(?= )` | 正向先行斷言 | `/\d+(?=px)/` → 匹配後面接 "px" 的數字 | `/\d+(?=px)/` // `\d+` 匹配數字，`(?=px)` 後面必須是 "px" | 零寬斷言 |
| `(?! )` | 負向先行斷言 | `/\d+(?!px)/` → 匹配後面不接 "px" 的數字 | `/\d+(?!px)/` // `\d+` 匹配數字，`(?!px)` 後面不能是 "px" | 零寬斷言 |
| `(?<= )` | 正向後行斷言 | `/(?<=\$)\d+/` → 匹配前面有 "$" 的數字 | `/(?<=\$)\d+/` // `(?<=\$)` 前面必須是 "$"，`\d+` 匹配數字 | 零寬斷言 |
| `(?<! )` | 負向後行斷言 | `/(?<!\$)\d+/` → 匹配前面沒有 "$" 的數字 | `/(?<!\$)\d+/` // `(?<!\$)` 前面不能是 "$"，`\d+` 匹配數字 | 零寬斷言 |
| `^` | 字串開頭 | `/^Hello/` → 匹配開頭的 "Hello" | `/^Hello/` // `^` 錨點（字串開頭），`Hello` 匹配字面字串 | 錨點 |
| `$` | 字串結尾 | `/world$/` → 匹配結尾的 "world" | `/world$/` // `world` 匹配字面字串，`$` 錨點（字串結尾） | 錨點 |
| `\b` | 符合單字邊界 | `/you\b/` → 符合 you，但**不符合** your | `/you\b/` // `you` 匹配 "you"，`\b` 單字邊界（後面不能接字母） | 邊界 |
| `\B` | 符合非單字邊界 | `/you\B/` → 符合 your，但**不符合** you | `/you\B/` // `you` 匹配 "you"，`\B` 非單字邊界（後面必須接字母） | 邊界 |
| `[ ]` | 字元集合 | `/[aeiou]/` → 匹配任意母音字母 | `/[aeiou]/` // `[aeiou]` 匹配 a、e、i、o、u 中的任一個 | 字元集合 |
| `[^ ]` | 反向集合 | `/[^aeiou]/` → 匹配非母音字母 | `/[^aeiou]/` // `[^aeiou]` 匹配不是 a、e、i、o、u 的任一字元 | 字元集合 |
| `\d` | 符合數字 | `/\d+/` → 只要有數字就符合 | `/\d+/` // `\d` 匹配數字（0-9），`+` 一次或多次 | 字元類別 |
| `\D` | 符合非數字 | `/\D+/` → 只要非 0~9 的字元都符合 | `/\D+/` // `\D` 匹配非數字，`+` 一次或多次 | 字元類別 |
| `\s` | 符合空白符 | `/\s+/` → 包含空格、換行符、定位符、換頁符 | `/\s+/` // `\s` 匹配空白字元（空格、Tab、換行等），`+` 一次或多次 | 字元類別 |
| `\S` | 符合非空白符 | `/\S+/` → 非空格、換行符、定位符、換頁符的內容 | `/\S+/` // `\S` 匹配非空白字元，`+` 一次或多次 | 字元類別 |
| `\w` | 包含數字、字母與底線 | `/\w+/` → 相當於 `[A-Za-z0-9_]` | `/\w+/` // `\w` 匹配單字字元（字母、數字、底線），`+` 一次或多次 | 字元類別 |
| `\W` | 不包含數字、字母與底線 | `/\W+/` → 相當於 `[^A-Za-z0-9_]` | `/\W+/` // `\W` 匹配非單字字元，`+` 一次或多次 | 字元類別 |
| `\t` | 符合定位符號（Tab 鍵） | `/\t+/` → 只要有定位符號就符合 | `/\t+/` // `\t` 匹配 Tab 鍵，`+` 一次或多次 | 特定字元 |
| `\n` | 換行符號 | `/\n+/` → 只要有換行符號就符合 | `/\n+/` // `\n` 匹配換行符（LF），`+` 一次或多次 | 特定字元 |
| `\r\n` | 回車與換行符號 | `/\r\n/` → 常用於 Windows 系統的換行 | `/\r\n/` // `\r` 回車符（CR），`\n` 換行符（LF） | 特定字元 |
| `.` | 點號，代表除換行符以外的任何字元 | `/.+/` → 匹配任意字元（除換行外） | `/.+/` // `.` 匹配任意字元（除換行），`+` 一次或多次 | 特定字元 |
