以下是 `split()` 和 `replace()` 的完整表格解說：

## split() 與 replace() 方法表格解說

### 一、split() 方法（字串分割）

將字串依照正規表達式或指定字元分割成陣列。

| 方法 | 所屬物件 | 語法 | 回傳值 | 受 `g` 影響 | 說明 |
|:---|:---:|:---|:---|:---:|:---|
| `split()` | String | `str.split(separator, limit)` | 陣列 (Array) | ❌ 否（忽略 `g`） | 依照分隔符將字串切割成陣列，可指定回傳最大長度 |

#### split() 範例與字元解說

| 範例 | 程式碼 | 回傳結果 | 字元解說 |
|:---|:---|:---|:---|
| 字串分割（字面字元） | `"a,b,c".split(",")` | `["a", "b", "c"]` | `split(",")` // 以逗號作為分隔符切割字串 |
| 字串分割（字面字元，限制長度） | `"a,b,c".split(",", 2)` | `["a", "b"]` | `split(",", 2)` // 以逗號切割，只取前 2 個元素 |
| 正規表達式分割（數字） | `"a1b2c3".split(/\d/)` | `["a", "b", "c", ""]` | `split(/\d/)` // `\d` 匹配數字，以數字為分隔符切割 |
| 正規表達式分割（空白） | `"hello world foo".split(/\s+/)` | `["hello", "world", "foo"]` | `split(/\s+/)` // `\s+` 匹配一個以上空白，以空白為分隔符 |
| 保留分隔符（捕獲組） | `"a1b2c".split(/(\d)/)` | `["a", "1", "b", "2", "c"]` | `split(/(\d)/)` // `(\d)` 捕獲組會保留分隔符在結果中 |
| 移除空字串（常見技巧） | `"a,b,c,".split(",")` | `["a", "b", "c", ""]` | `split(",")` // 結尾逗號會產生空字串元素，可用 `filter(Boolean)` 過濾 |

### 二、replace() 方法（字串替換）

將符合正規表達式的部分替換成指定字串或透過函數處理。

| 方法 | 所屬物件 | 語法 | 回傳值 | 受 `g` 影響 | 說明 |
|:---|:---:|:---|:---|:---:|:---|
| `replace()` | String | `str.replace(regexp, replacement)` | 新字串 | ✅ 是（無 `g` 只替換第一個） | 將匹配到的內容替換為新字串，支援 `$1`、`$2` 等捕獲組參照 |

#### replace() 基本範例與字元解說

| 範例 | 程式碼 | 回傳結果 | 字元解說 |
|:---|:---|:---|:---|
| 字面字串替換 | `"hello world".replace("world", "JS")` | `"hello JS"` | `replace("world", "JS")` // 將 "world" 替換為 "JS" |
| 正規表達式替換（無 `g`） | `"a1b2c".replace(/\d/, "x")` | `"axb2c"` | `replace(/\d/, "x")` // `\d` 匹配第一個數字 "1"，替換為 "x" |
| 正規表達式替換（有 `g`） | `"a1b2c".replace(/\d/g, "x")` | `"axbxc"` | `replace(/\d/g, "x")` // `\d` 匹配所有數字，全部替換為 "x" |

#### replace() 捕獲組範例（`$1`, `$2` 用法）

| 範例 | 程式碼 | 回傳結果 | 字元解說 |
|:---|:---|:---|:---|
| 交換位置 | `"John Doe".replace(/(\w+) (\w+)/, "$2, $1")` | `"Doe, John"` | `/(\w+) (\w+)/` // 兩個捕獲組分別匹配名和姓，`$2, $1` 交換順序 |
| 日期格式轉換 | `"2026-04-13".replace(/(\d{4})-(\d{2})-(\d{2})/, "$2/$3/$1")` | `"04/13/2026"` | `$1` 年，`$2` 月，`$3` 日，改為月/日/年格式 |
| 添加前後綴 | `"hello".replace(/(hello)/, "【$1】")` | `"【hello】"` | `$1` 參照捕獲組的 "hello"，前後加上括號 |
| 使用命名捕獲組 | `"2026-04-13".replace(/(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/, "$<month>/$<day>/$<year>")` | `"04/13/2026"` | `$<month>` 等參照命名捕獲組，較 `$1` 更易讀 |

#### replace() 特殊替換符號表

| 符號 | 說明 | 範例 | 結果 |
|:---:|:---|:---|:---|
| `$$` | 插入一個 `$` 符號 | `"100".replace(/\d+/, "$$100")` | `"$100"` |
| `$&` | 插入完整匹配的字串 | `"hello".replace(/ll/, "$&$&")` | `"hellllo"` |
| `` $` `` | 插入匹配位置**之前**的字串 | `"hello".replace(/ll/, "$`")` | `"heheo"` |
| `$'` | 插入匹配位置**之後**的字串 | `"hello".replace(/ll/, "$'")` | `"heoo"` |
| `$n` | 插入第 n 個捕獲組（n=1~9） | `"a1b".replace(/(\d)/, "$1$1")` | `"a11b"` |
| `$<name>` | 插入命名捕獲組 | `"2026".replace(/(?<year>\d+)/, "$<year>年")` | `"2026年"` |

#### replace() 使用回呼函數（進階用法）

| 範例 | 程式碼 | 回傳結果 | 字元解說 |
|:---|:---|:---|:---|
| 數字加倍 | `"a1b2c3".replace(/\d/g, (m) => m * 2)` | `"a2b4c6"` | `(m) => m * 2` // 匹配到的每個數字乘以 2 |
| 格式化數字 | `"price: 100".replace(/\d+/, (m) => "$" + m)` | `"price: $100"` | `(m) => "$" + m` // 在數字前加上 "$" |
| 使用捕獲組（函數參數） | `"2026-04-13".replace(/(\d{4})-(\d{2})-(\d{2})/, (match, year, month, day) => `${month}/${day}/${year}`)` | `"04/13/2026"` | 函數參數依序為：完整匹配、第1捕獲組、第2捕獲組... |

#### replace() 回呼函數參數說明

| 參數 | 說明 | 範例值 |
|:---|:---|:---|
| `match` | 完整匹配的字串 | `"2026-04-13"` |
| `p1, p2, ...` | 第1個、第2個...捕獲組 | `"2026"`, `"04"`, `"13"` |
| `offset` | 匹配位置在原始字串中的索引 | `0` |
| `string` | 原始輸入字串 | `"2026-04-13"` |
| `groups` | 命名捕獲組物件（若有） | `{year:"2026", month:"04", day:"13"}` |

```javascript
// 完整參數範例
"2026-04-13".replace(/(?<year>\d{4})-(\d{2})-(\d{2})/, (...args) => {
  console.log(args)
  // 輸出：["2026-04-13", "2026", "04", "13", 0, "2026-04-13", groups: {year:"2026"}]
})
```

### 三、split() vs replace() 對照表

| 比較項目 | `split()` | `replace()` |
|:---|:---|:---|
| 所屬物件 | String | String |
| 主要用途 | 將字串切割成陣列 | 將字串中的內容替換 |
| 回傳值 | 陣列 (Array) | 新字串 (String) |
| 是否改變原字串 | ❌ 否（回傳新陣列） | ❌ 否（回傳新字串） |
| 受 `g` 旗標影響 | ❌ 否（忽略 `g`） | ✅ 是（無 `g` 只替換第一個） |
| 捕獲組影響 | 有捕獲組時會保留分隔符 | 可用 `$1`、`$2` 參照捕獲組 |
| 支援回呼函數 | ❌ 否 | ✅ 是 |
| 常見應用 | CSV 解析、字串分割處理 | 文字取代、日期格式化、模板替換 |

```javascript
// 實際應用對比
const str = "a1b2c3"

// split：取出所有字母
str.split(/\d+/)        // ["a", "b", "c", ""]

// replace：將數字替換為逗號
str.replace(/\d/g, ",") // "a,b,c,"
```
