# RegExp `u` 旗標（Unicode 模式）核心解析

## 一句話總結

> **`u` 旗標將正規表示式的比對單位，從「16 位元（Code Unit）」轉為「完整字元（Code Point）」**

---

## 1. 預設模式 vs `u` 模式對照表

| 項目 | 預設模式（無 `u`） | Unicode 模式（有 `u`） |
|------|-------------------|------------------------|
| 比對單位 | 16 位元（Code Unit） | 完整 Unicode 字元（Code Point） |
| `.` 的意義 | 一個 16 位元 | 一個完整字元 |
| 字元長度判斷 | 以 UTF-16 為單位 | 以 Unicode 碼點為單位 |
| Emoji「😀」長度 | 2 | 1 |
| 罕見字「𠮷」長度 | 2 | 1 |

### 範例：`/^.$/` 的行為差異

**語法解釋：**

- `^`：字串開頭
- `.`：任何單一字元（預設模式為 16 位元單位，`u` 模式為完整 Unicode 字元）
- `$`：字串結尾

合起來表示「整個字串必須是『一個』字元」。

| 測試字元 | 預設模式 `/^.$/` | `u` 模式 `/^.$/u` |
|----------|-----------------|-------------------|
| `"A"`（一般英文字） | ✅ true | ✅ true |
| `"我"`（中文字，BMP 範圍） | ✅ true | ✅ true |
| `"😀"`（Emoji，增補範圍） | ❌ false | ✅ true |
| `"𠮷"`（罕見古字，增補範圍） | ❌ false | ✅ true |

> **原因**：`😀` 與 `𠮷` 在 UTF-16 編碼中長度為 2，預設模式的 `.` 只能比對 1 個 16 位元單位，所以失敗；`u` 模式下 `.` 比對完整 Unicode 字元，因此成功。

---

## 2. Unicode 編碼表示法：`\uXXXX` vs `\u{XXXXX}`

| 範圍 | 碼點範圍 | 表示法 | 是否需要 `u` 旗標 | 範例 |
|------|----------|--------|------------------|------|
| 基本範圍（BMP） | `0000` ~ `FFFF` | `\uXXXX`（4 位） | 不需要 | `\u6211` → `"我"` |
| 增補範圍（Supplementary） | `10000` 以上 | `\u{XXXXX}`（5~6 位） | **需要** | `\u{1F600}` → `"😀"` |

### 範例：正確與錯誤寫法對照

| 目標字元 | 正確寫法 | 錯誤寫法 | 錯誤原因 |
|----------|----------|----------|----------|
| `"我"`（U+6211） | `/\u6211/` 或 `/\u6211/u` | — | — |
| `"😀"`（U+1F600） | `/\u{1F600}/u` | `/\u{1F600}/`（無 u） | `{}` 被當成普通字元 |
| `"😀"`（U+1F600） | `/\u{1F600}/u` | `/\uD83D\uDE00/`（代理對） | 不易閱讀與維護 |

> **補充**：`\uD83D\uDE00` 是 `"😀"` 的 UTF-16 代理對（Surrogate Pair），雖然不靠 `u` 也能比對，但寫法冗長、容易出錯。

---

## 3. 實用建議

| 情境 | 建議 |
|------|------|
| 內容包含 Emoji | ✅ 使用 `u` 旗標 |
| 內容包含罕見古字（如「𠮷」、「𪚥」） | ✅ 使用 `u` 旗標 |
| 內容為純英文 / 數字 / 基本符號 | ⚠️ 可不加，但加無壞處 |
| 需要正確計算字元長度或進行逐字比對 | ✅ 使用 `u` 旗標 |
| 使用 `\u{...}` 寫法 | ✅ **必須**加上 `u` 旗標 |

---

## 4. 總結表

| 重點 | 說明 |
|------|------|
| `u` 旗標的核心 | 將比對單位從「16 位元」提升為「完整 Unicode 字元」 |
| 影響最明顯的場合 | 處理 Emoji、罕見漢字（碼點 > `FFFF`） |
| 常見錯誤 | 忘記加 `u` 導致 `\u{1F600}` 變成普通字串 |
| 最佳實務 | 處理非純英文內容時，**一律加上 `u` 旗標** |

---

## 5. 快速測試（可直接貼到瀏覽器主控台）

```javascript
// 測試 Emoji
/^.$/.test("😀")   // false
/^.$/u.test("😀")  // true

// 測試增補漢字
/^.$/.test("𠮷")   // false
/^.$/u.test("𠮷")  // true

// 測試 \u{} 寫法
/\u{1F600}/u.test("😀")   // true
/\u{1F600}/.test("😀")    // false（無 u 旗標）
